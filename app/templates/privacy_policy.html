隐私政策

最后更新日期：2024年5月28日

1. 我们如何收集和使用您的信息

陆梧GeoCo（以下简称"我们"）尊重并保护所有使用我们服务的用户的个人隐私权。本政策阐述了我们如何收集、使用和分享您的个人信息。

我们会出于以下目的收集和使用您的信息：

  • 用户注册与登录：当您注册陆梧GeoCo账户时，我们会收集您的手机号码，用于创建账户、身份验证以及后续的账户管理。如果您选择使用账号密码登录，我们会收集您的用户名和密码。
  • 地理编码服务：当您使用地理编码、逆地理编码、坐标转换等服务时，您提供的地址、位置信息将被用于处理您的请求，以提供准确的地理空间数据。这些信息可能在处理过程中通过加密方式发送给第三方地图服务提供商（如高德地图、百度地图、天地图等）。
  • 智能搜索与大模型：当您使用智能搜索或涉及大模型（LLM）的功能时，您的查询关键词和相关上下文信息可能会被发送给第三方AI服务提供商（如智谱AI等），以便获取更智能的分析结果。
  • API Key配置：如果您选择在设置中配置自己的地图或AI API Key，我们会存储您的这些Key，并优先用于您的请求。这些Key仅用于技术调用，不会被分享给无关第三方。
  • 积分管理：我们会记录您的积分余额、充值和消耗记录，以管理您的账户权益。
  • 意见反馈：当您提交意见反馈时，我们会收集您提供的描述、联系方式（如可选）以及您上传的图片，用于改进服务和解决您的问题。
  • 服务优化与安全：我们可能会收集您的设备信息、操作日志、IP地址等，用于分析服务使用情况、优化用户体验、保障服务安全和防范欺诈行为。

2. 信息共享与披露

我们不会向任何无关第三方出售、出租、分享或以其他方式披露您的个人信息，除非获得您的明确同意，或为以下目的所必需：

  • 履行服务所需：为完成您请求的地理编码、智能搜索等服务，我们可能将必要的匿名化或假名化信息分享给第三方地图或AI服务提供商。这些提供商受其自身隐私政策约束，我们建议您查阅其政策。
  • 法律法规要求：根据法律法规的规定，或行政、司法机关的要求，我们可能会向相关部门提供您的信息。
  • 保护合法权益：为保护我们、其他用户或公众的合法权益，我们可能会披露您的信息。

3. 信息安全

我们采用合理的物理、技术和管理措施来保护您的个人信息，防止数据被未经授权的访问、披露、使用、修改、损坏或丢失。例如，我们使用加密技术保护数据传输，并对数据进行匿名化处理以降低识别风险。

尽管我们已采取合理措施，但互联网并非绝对安全的环境，请您务必注意保护您的个人信息安全，不要随意向他人透露您的账户密码、验证码等敏感信息。

4. 您的权利

在适用法律允许的范围内，您可能拥有以下与您的个人信息相关的权利：

  • 访问权：您有权访问我们持有的您的个人信息。
  • 更正权：当您发现我们处理的关于您的个人信息有错误时，您有权要求我们进行更正。
  • 删除权：在某些情况下，您有权要求我们删除您的个人信息。
  • 注销账户：您可以联系我们注销您的陆梧GeoCo账户。账户注销后，我们将停止为您提供产品和服务，并删除您的个人信息（法律法规另有规定的除外）。

如您需要行使上述权利，请通过本政策提供的联系方式与我们取得联系。我们将在合理的时间内处理您的请求。

5. 未成年人保护

陆梧GeoCo服务主要面向成年用户。如果您的年龄未满18周岁，请在监护人的陪同下阅读本政策，并在获得监护人同意的前提下使用我们的服务或向我们提供信息。

如果您是未成年人的监护人，当您对您所监护的未成年人的个人信息有相关疑问时，请及时与我们联系。

6. 政策更新

我们可能会不时更新本隐私政策。任何更新都将在本页面发布，并注明更新日期。我们建议您定期查看本政策，以了解我们如何保护您的信息。
