    陆梧GeoCo 使用教程
    快速上手
    1. 登录账户（未登录将无法开始编码）。
    2. 将地址粘贴到输入框，每行一个地址。
    3. 点击"多源编码"开始获取候选结果。
    4. 点击"智能编码"可一键完成编码并自动校准。
    5. 在批量结果中确认合适结果，或进入"逐条查看与校准"。
    6. 确认后，点击"导出文件"下载为 Excel/KML/SHP。
    常用功能
    • 数据清洗：支持自动分行、统一省市前缀、自动补全行政区划。
    • 地址查找工具：在地图上搜索并选择更精准的地点。
    • API Key 配置：在"设置"中填写各地图/AI Key，可提升额度与体验。
    • 积分与充值：右上角可查看积分，支持充值与活动获取。
