GeoCo UI 架构分层级中英对照名称 (Updated Architecture)
I. 全局布局与导航 (Global Layout & Navigation)
    1.1 顶部用户状态与操作区 (Top User Status & Action Bar)
    top-user-bar (Class for the overall bar)
    未登录状态 (Unsigned User State): user-unsigned (ID)
    已登录状态 (Signed-in User State): user-signed (ID)
    登录按钮 (Login Button): showLoginModalBtn (ID)
    免费赚积分按钮 (Earn Free Points Button): showFreePointsModalBtn (ID)
    积分充值按钮 (Recharge Points Button): showRechargeModalBtn (ID)
    通知铃铛 (Notification Bell)
    用户名显示 (Username Display): username-display (ID)
    用户下拉菜单 (User Dropdown Menu): userDropdown (ID)
II. 核心功能模块 (Core Function Modules)
(这些模块在主内容区 (.main-container .bottom-section) 内动态切换显示)
    2.1 地址输入与主要编码区 (Address Input & Main Geocoding Area)
        address-input-section (Class, formerly top-section)
        地址输入框 (Address Textarea): addresses (ID)
        高级选项折叠面板 (Advanced Options Collapse Panel): advancedAddressOptions (ID)
        地址自动处理按钮 (Auto-Complete Address Button): autoCompleteBtn (ID)
        地名类型输入与标签 (Location Type Input & Tags): locationType (ID), locationTypeTags (ID)
        普通地理编码按钮 (Normal Geocode Button): normalGeocodeBtn (ID)
        一键智能编码按钮 (One-Click Smart Geocode Button): smartGeocodeBtn (ID)
    2.2 批量结果总览与管理区 (Batch Results Overview & Management Area)
        cascadeResultsContainer (ID)
        结果总览地图 (Results Overview Map): resultsOverviewMap (ID)
        批量处理结果表格 (Batch Results Table): cascadeResultsBody (ID for tbody)
        逐条查看与校准入口按钮 (View Details Sequentially Button): viewDetailsSequentiallyBtn (ID)
        导出文件下拉菜单 (Export Files Dropdown): exportExcelBtn, exportKmlBtn, exportShpBtn (IDs)
        更多操作下拉菜单 (More Actions Dropdown): autoSelectAllBtn, smartSearchAllBtn (IDs)
        保存结果到账户按钮 (Save Results to Account Button): saveResultsBtn (ID)
    2.3 逐条校准区 (Sequential Calibration Area)
        detailedReviewSection (ID)
        地图校准面板 (Map Calibration Panel): itemCalibrationMap (ID)
        手动标注按钮 (Manual Mark Button): manualMarkBtnOnMap (ID)
        结果详情与操作面板 (Result Details & Operations Panel)
        原始地址信息显示 (Original Address Display): originalAddressDisplay (ID)
        当前选定结果详情卡片 (Selected Result Detail Card): selectedResultDetailCard (ID)
        结果导航 (Result Navigation): prev-result-btn, next-result-btn (Classes)
        各平台结果与操作折叠面板 (API Results & Operations Collapse Panel): advancedApiResultsContainer (ID)
        calibration-controls-panel (Class, formerly operation-area)
        逐一校准按钮 (Calibrate One-by-One Button): batchGeocodeBtn (ID)
        智能匹配按钮 (Single Intelligent Match Button): singleIntelligentMatchBtn (ID)
        各平台结果卡片 (API Result Cards): api-result-card (Class, with data-api attributes)
        地址查找工具入口按钮 (Address Search Tools Button): showAddressSearchToolsBtn (ID)
    2.4 地址查找工具区 (Address Search Tools Area)
        addressSearchToolsContainer (ID)
        2.4.1 POI 地图搜索面板 (POI Map Search Panel)
        poi-search-pane (ID for Tab Content)
        poi-search-panel (Class, formerly map-search-grouping)
        地图搜索输入框 (Map Search Input): mapSearchInput (ID)
        地图搜索按钮 (Map Search Button): mapSearchBtn (ID)
        2.4.2 智能地址情报分析面板 (Smart Address Intelligence Analysis Panel)
        smart-intelligence-pane (ID for Tab Content)
        smart-intelligence-panel (Class, formerly smart-search-container)
        智能搜索输入框 (Smart Search Input): searchInput (ID)
        智能搜索按钮 (Smart Search Button): smartSearchForm (ID for form)
